# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Documentation and discoverability assets (README, examples, article, changelog).

### Changed

- npm package renamed from `recursive-yml` to `yaml-dot-resolve`.

## [0.2.1] - 2026-04-14

### Changed

- Package metadata (`repository` URL and related npm fields).

## [0.2.0] - 2026-04-14

### Added

- Recursive `${...}` substitution for YAML documents with `custom.stage` injection, nested placeholders, optional `:-` defaults, and Vitest coverage.
