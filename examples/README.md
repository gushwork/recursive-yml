# Examples

Runnable samples that use `yaml-dot-resolve`. For published versions, replace the `file:` dependency with a normal semver range from npm.

| Directory        | Description                                      |
| ---------------- | ------------------------------------------------ |
| [`stage-config`](stage-config) | Stage name drives `custom.<stage>` lookup. |

## Try locally

From the **repository root**:

```bash
npm install
npm run build
cd examples/stage-config
npm install
npm start
```
